5. Uso de formatações

O negrito já foi bastante utilizado nesse código, com ele usamos as tags STRONG ou B.

No caso das palavras em Itálico/Ênfase usamos as tags EM, I;

Quando queremos marcar uma palavra usamos a tag MARK. Podemos editar sua cor no CSS.

Se for precisar destacar uma palavra em uma frase podemos usar a tag BIG ou a tag SMALL.

Podemos marcar uma palavra como excluida para indicar que ele deve ser lido, mas não considerado. Usamos a tag DEL.

Podemos marcar uma palavra como inserida para dar uma ênfase e indicar que ele foi adicionada depois. Usamos a tag INS.

Usamos o texto sobrescrito para inserir coisas do tipo: x2+310. Usamos a tag SUP.

Usamos o texto subscrito para inserir coisas do tipo:H2O. Usamos a tag SUB.
